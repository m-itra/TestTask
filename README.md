# Поиск минимальной IP-сети

Утилита определяет наименьшую общую сеть для двух IP-адресов.
Поддерживаются адреса форматов IPv4 и IPv6.

## Запуск

#### С аргументами командной строки:

IPv4:

```bash
python main.py 192.168.1.10 192.168.1.20
```

Результат:

```text
192.168.1.0/27
```

#### Без аргументов командной строки адреса можно ввести по ходу работы:

```bash
python main.py
```
